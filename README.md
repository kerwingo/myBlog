# Kerwin Blog

> 新一代 Vue2 + Nuxt.js + Semantic UI 搭建的个人博客，支持服务端渲染。

## 链接

http://blog.kerwin.plus

## 特征

- 服务端渲染

- 支持移动端

- 支持暗黑模式、圆滑和锐利模式。

- 代码拆分每一个 page

- 仅加载关键 css（页面级）

- 自定义底部等

- 路由平滑切换处理
